public subroutine exportQEInput()

Uses

    • miscUtilities

Create QE input files for all different modes by copying all of the QEInput except the newAtomicPositions for each mode

Walkthrough

  • For each mode between modeI and modeF
  • If the folder for this mode doesn't already exist, make it
  • Copy the QEInput file into the new folder, changing the positions to be the newAtomicPosition
